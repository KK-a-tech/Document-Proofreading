校正くん(仮)

本コンテンツは校正支援ツールとしてご利用ください。
「yahoo!テキスト解析 Web API 校正支援(V2)」を用いた文章校正機能をご提供しております。

※入力したデータは入力した情報はYahoo! JAPANへ送信されます。通信は暗号化されていますが、Yahoo! JAPANに保存される可能性はありますのでご注意ください。

検査内容

  • 表記・表現の間違い、不適切な表現の検出
    … 誤変換、誤用、使用注意語、不快語 (使用不適切な語や隠語など)、機種依存文字または拡張文字、外国地名、固有名詞、人名、ら抜き言葉
  • わかりやすい表記にするための提案
    … 当て字、表外漢字、用字
  • 文章をよりよくするための提案
    … 用語言い換え、二重否定、助詞不足の可能性あり、冗長表現、略語

文章入力フォーム

フォームに入力し、「校正開始」ボタンをクリックしてください。

校正案

校正案を表示しております。
校正案から修正したい箇所をチェックボックスより修正ください。
全て修正したい場合は「一括承認」をご利用ください。

校正結果出力

「コピー」ボタンをクリックすると校正後の文章をクリップボードにコピーします。
